# POLITICA DE PRIVACIDADE - TRILAB TREINAMENTO

A TriLab TREINAMENTO respeita a privacidade dos usuarios e trata os dados pessoais em conformidade com a **Lei Geral de Protecao de Dados (Lei no 13.709/2018 - LGPD)**.

---

## 1. DADOS COLETADOS

Podemos coletar os seguintes dados:

### Dados cadastrais

* nome
* email
* sexo
* idade

### Dados de treinamento

* frequencia de treinos
* feedback de esforco
* historico de lesoes
* relato de dor ou desconforto
* progresso de treino

Esses dados sao utilizados exclusivamente para personalizar a experiencia de treinamento.

---

## 2. DADOS SENSIVEIS

Algumas informacoes fornecidas pelo usuario podem ser consideradas **dados sensiveis**, como:

* informacoes sobre dor
* historico de lesoes
* limitacoes fisicas

Esses dados sao coletados apenas com **consentimento do usuario** e utilizados exclusivamente para melhorar a prescricao de treino.

---

## 3. FINALIDADE DO USO DOS DADOS

Os dados coletados sao utilizados para:

* gerar treinos personalizados;
* adaptar exercicios conforme feedback do atleta;
* gerar relatorios de desempenho;
* permitir acompanhamento por treinadores vinculados;
* melhorar os servicos da plataforma.

---

## 4. COMPARTILHAMENTO DE DADOS

Os dados podem ser compartilhados apenas nas seguintes situacoes:

* com treinadores vinculados autorizados pelo atleta;
* com prestadores de servico essenciais ao funcionamento da plataforma (hospedagem, pagamento);
* mediante obrigacao legal.

A TriLab TREINAMENTO nao vende dados pessoais.

---

## 5. ARMAZENAMENTO DOS DADOS

Os dados sao armazenados em servidores seguros e protegidos por medidas tecnicas e administrativas adequadas.

---

## 6. DIREITOS DO TITULAR DOS DADOS

O usuario possui os seguintes direitos:

* acessar seus dados;
* corrigir dados incorretos;
* solicitar exclusao de dados;
* revogar consentimento.

Solicitacoes podem ser feitas pelo email:

**[suporte@trilabtreinamento.com](mailto:suporte@trilabtreinamento.com)**

---

## 7. COOKIES

A plataforma pode utilizar cookies para melhorar a experiencia do usuario.

---

## 8. TEMPO DE RETENCAO

Os dados sao mantidos enquanto o usuario possuir conta ativa ou enquanto necessario para cumprimento de obrigacoes legais.
